# mamosuku_skill-trait_sheet
TRPG「マモノスクランブル」（著者：からすば晴）のマギと特性を効率的に閲覧できるサイトです。

# 機能

1. マギと特性の一覧表示
2. マギの絞り込み・並び替え
3. 使用するマギの情報を即座に取得
